groundUP!

Roles: UX Researcher, Front-End Dev
Duration: December 2014 - Current

Description
groundUP! is a web application that aims to motivate users to make daily progress towards their long term goals by helping them see the big goal as multiple, smaller tasks. It encourages users to make each day productive and to gain confidence and be more self motivated as they become closer towards their goal.

Currently a work in progress; details upon request.
